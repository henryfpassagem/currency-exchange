# 💱 Currency Exchange Application

A lightweight, responsive web application that fetches and displays historical currency exchange rates over a 30-day period. Built using pure vanilla frontend technologies and integrated with an external financial API.

## ✨ Features

* **Real-Time Data Fetching:** Interacts dynamically with the AwesomeAPI to fetch currency conversions.
* **30-Day Historical Trends:** Converts API data streams into readable daily listing parameters.
* **Dark-Themed Modern UI:** Custom styles tailored with a sophisticated CSS palette.
* **Cross-Browser Compatibility:** Designed using clean native JavaScript modules and standard relative fallbacks for smooth GitHub Pages deployment.

---

## 🛠️ Tech Stack

* **HTML5** - Semantic layout and structure
* **CSS3** - Custom properties (variables), Flexbox, and Grid layouts
* **Vanilla JavaScript** - Document Object Model (DOM) manipulation and asynchronous `Fetch API` data streaming
* **SweetAlert2** - Beautiful customized popup modals for error handling
* **AwesomeAPI** - Free economic and financial currency exchange REST API wrapper
